Welcome to this Unix shell command cheat sheet!

Ready to learn some things about Unix shell command? Well here we go!

Here are a few Unix shell commands for your programming pleasure :)

Shell Command Names Description
ls lists files
tree goes to the beginning of your directory and prints the files
file <filename> tells you what type of file the named file is.
type <commandname> tells you what type of command the given command is
apropos <commandname> tells you if the given command is downloaded
man brings up the manual pages
cd <directoryname> moves you to the named directory
mkdir <directoryname> creates a new directory with the given name
pwd prints the working directory
echo <argument> prints the given arguments on the screen
cat <string> puts strings together and prints them on the screen
less <argument> prints the given arguments on the screen
